# intrinsic_plast
Code used for Intrinsic Plasticity Underlies the Malleability of Neural Network Heterogeneity (https://doi.org/10.1101/2025.06.09.658695)

Functions: contains the main functions used to simulate the network, intrinsic plasticity, etc.

Plotting functions: not yet complete, but contains the code for the layouts/plotting of some of the figures
